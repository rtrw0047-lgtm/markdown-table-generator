# Markdown Table Generator

Visually create and edit markdown tables with copy-ready output.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Generator** (or Skip).

**Step 3:** Click any cell in the table to edit its content. The default table has Name, Age, City columns.

**Step 4:** Use the buttons to modify the table:
- **+ Row** - add a new row
- **+ Column** - add a new column
- **- Row** - remove the last row
- **- Column** - remove the last column

**Step 5:** The markdown output updates automatically below the table.

**Step 6:** Click **Copy Markdown** to copy the table code to your clipboard.

**Step 7:** See the HTML preview to check how the table looks when rendered.

## Features
- Visual table editing (click to edit cells)
- Add/remove rows and columns
- Auto-generated markdown syntax
- Live HTML preview
- Copy markdown to clipboard
- Column width auto-alignment

## Tech
HTML + CSS + JavaScript | Clipboard API
